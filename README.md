# 关于项目

Python爬取豆瓣电影top250 使用flask框架可视化数据

> 项目原地址：https://github.com/lzz110/douban_movies_top250

项目技术栈：`Flask框架、Echarts、WordCloud、SQLite`

环境：`Python3`

## 目录结构

```html
│  app.py				----- flask框架 文件
│  movie.db				----- 数据库
│  README.md
│  requirements.txt		----- 依赖包环境版本
│  spider.py			----- 爬取数据 文件
│  testCloud.py			----- 词云生成 文件
├─static				----- 静态页面
├─templates				----- HTML页面
└─venv					----- 虚拟环境
```

## 本地使用

```html
$ pip install -r requirements.txt
$ python app.py
```
